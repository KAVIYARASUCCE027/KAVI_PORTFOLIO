# Kaviyarasu M - Developer Portfolio

A premium Awwwards-style developer portfolio built with Next.js 15, featuring dark futuristic design, glassmorphism, and smooth animations.

## Tech Stack

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - UI animations
- **GSAP** - Hero animations
- **Swiper.js** - Project carousel
- **React Three Fiber** + **Drei** + **Three.js** - 3D hero scene
- **Lenis** - Smooth scrolling
- **Lucide React** - Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Project Structure

```
app/           # Next.js app router
components/    # Reusable UI components
sections/      # Page sections
hooks/         # Custom React hooks
utils/         # Utilities and data
public/        # Static assets
```

## Customization

- Update personal data in `utils/data.ts`
- Replace profile image in `sections/Hero.tsx`
- Add resume PDF to `public/resume.pdf`
- Update social links in `utils/data.ts`
